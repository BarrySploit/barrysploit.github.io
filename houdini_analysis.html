Houdini Worm Analysis

Analyze the behavior and impact of the Houdini Worm

About This Project

Found in a USB Docking Station bought off of Amazon, This analysis of the Houdini Worm (angry birds.vbe) aims to provide insights into the behavior, propagation, and impact of the notorious Houdini Worm.

Key Details

  • Worm Name: Houdini Worm
  • Type: Malware
  • Analysis: Ongoing
  • Researcher: Jason "BarrySploit" Barry

A good buddy of mine had ordered a USB docking station off of Amazon from a seemingly reputable distributor. The docking station came with a small USB that was immediately flagged by Windows Defender. Once opened, the USB contained the following at the root of the drive: 2 shortcuts (lnk files) 2 hidden folders 1 hidden vbe script

The folders and the vbe script were not only marked as hidden, but also marked as system files. This meant that even with "Show hidden files" enabled, you still can't see the files. There is a specific setting in file explorer to allow you to see system files. (See below). Go to File Explorer, View, Options, advanced and unselect this box.
